应届生的程序员之路:
 * 应届生如何选择技术
  * Java - 最多
    * JavaEE - 服务器端
      * 行业发展久
      * 相对稳定
      * 人员需求稳定
      * 入门薪资相对低
    * Android - 移动端
      * 09年进入国内
      * 3年以上经验
  * C - 入门
  * C++ - 游戏开发
  * C# - 服务器端不多
  * .NET - WP开发
  * javascript 脚本语言  HTML5标准

  * 建议:
    * 任何语言对于我们来讲,都是一样的.
    * 首先选择市场需求比较大.
    * 其次选择相对来讲比较容易(入门) - Java
 * 如何学好软件开发
   * 初级学知识,高级的学规律.
   * 一定要练. - 抄来的都是别人的,写来的都是自己的!
     * 抄 - 1)记录所有的错误;2)对比别人的代码
     * 留注释 - 1)记录所有的错误;2)除了这个思路之外,还有没有其他的方法
     * 留需求 - 真正地开始敲代码 - 程序员
   * 看视频 - ？ 【看代码:struts2 Demo】
 * 如何应对技术面试
   * 面试题:
     * 搜面试题 - 2000道以上.
     * 购买面试书籍
     * 求助身边的人
   * 面试的形式:
     * 笔试:
       * 形式:考试 - 技术性
     * 面试
       * 技术性 - 技术能力
       * 非技术性 - 综合能力
   * 如何处理面试题:
     * 将面试题,分为两份.
     * 其中一份,将所有答案清除掉.
     * 闭卷答题
     * 对答案
     * 答对不要管了,只管答错的.
   * 非技术性:
     * 沟通能力
     * 紧张情绪
     * ...
 * 如何看待第一份工作
   * 程序员成长路线 - 放长线钓大鱼.
   * 更多是学习,不是赚钱.
     * 学技术 - 技术上的积累
     * 学经验 - 工作经验【为人处事】
   * 不要看公司可以给你提供怎么样的待遇,要看公司可以提供给你什么样的平台.
 * 怎样变成主力
   * 一屋不扫,何以扫天下.
   * 适当表现.
   * 承担责任.
   * 团队内部很多人喜欢你.
 * 寻求可持续发展
   * 如何从一个程序员晋升到经理
     * 定目标
     * 罗列目标所需要的能力
     * 逻辑目前所掌握的能力
     * 做差 - 找到你目前还没有掌握的能力
